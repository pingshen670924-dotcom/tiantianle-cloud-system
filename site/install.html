天天樂手機雲端獨立版

使用 GitHub Pages 與 GitHub Actions，電腦關機也能從手機開啟。

登入 GitHub 後立即雲端更新

真正手機網址

https://pingshen670924-dotcom.github.io/tiantianle-cloud-system/

手機安裝必須使用這個 GitHub Pages 網址，不是電腦的 file:/// 路徑。

Android / Chrome

  1. 手機開 GitHub Pages 網址。
  2. 點上方一鍵啟動天天樂雲端版。
  3. 瀏覽器若跳出安裝提示，選擇安裝 App。
  4. 安裝後從手機主畫面開啟天天樂。

iPhone / Safari

  1. 點 Safari 分享按鈕。
  2. 選擇「加入主畫面」。
  3. 完成後從主畫面開啟天天樂。

這是雲端版入口：更新由 GitHub Actions 執行，畫面由 GitHub Pages 提供。手機不需要連回電腦。
