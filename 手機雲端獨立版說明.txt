天天樂手機雲端獨立版

你要的手機版不是 file:///C:/... 本機檔案。
真正免電腦手機版必須先上線到 GitHub Pages，手機開的是 https://帳號.github.io/tiantianle-cloud-system/。

第一次上線：
1. 在電腦雙擊「天天樂雲端一鍵上線.bat」。
2. 依畫面完成一次 GitHub 官方登入。
3. 系統會建立 tiantianle-cloud-system 程式庫。
4. 系統會上傳完整天天樂主程式、歷史資料庫、戰報與手機網站。
5. 系統會啟用 GitHub Pages 與 GitHub Actions。
6. 第一次雲端更新完成後，會產生「天天樂手機雲端網址.txt」。

之後手機獨立使用：
1. 手機開「天天樂手機雲端網址.txt」裡的網址。
2. Android 用 Chrome 安裝 App。
3. iPhone 用 Safari 分享，選「加入主畫面」。
4. 之後電腦關機也能開啟。
5. 每日更新由 GitHub Actions 自動執行。
6. 手機要手動立即更新時，按頁面上的「登入 GitHub 後立即雲端更新」。

注意：
- GitHub Pages 網址才是真正雲端版。
- file:///C:/... 只是電腦本機預覽，不是手機獨立版。
- 第一次上線通常需要 5 到 10 分鐘。
